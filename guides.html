FAQ

Questions frequentes

Ce site est-il relie officiellement au jeu ?

Non, cette version est une base fan-made de style wiki. Elle peut servir de projet personnel, portfolio ou base communautaire.

Peut-on l'ouvrir sans serveur local ?

Oui. Il suffit d'ouvrir les fichiers HTML dans ton navigateur pour naviguer sur le site.

Peut-on ajouter de vraies donnees du jeu plus tard ?

Oui, le site est fait pour ca. On pourra remplacer chaque carte par les vraies infos de Sailor Piece quand tu voudras.
